Simple-jQuery-UA-Spoofing

Check the source of your document, it should have 3 classes attached to the element you initialized the plugin with.

  • OS: win(dows), mac, lin(ux), idevice(ipod, ipad, iphone)
  • Browser: f(ire)f(ox), c(h)r(o)m(e), s(a)f(a)r(i), w(e)bk(it)
  • Browser + Version: a combination of the brower class and the current version, for instance "ff40" for Firefox 4.

A sample class would be: "win ff ff40", that stands for Windows, Firefox 4
