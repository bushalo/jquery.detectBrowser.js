jquery.detectBrowser.js for jQuery 1.0+

Check the source of your document, it should have 3 classes attached to the element you initialized the plugin with.

  • OS: win, mac, lin, idevice.
  • Browser: ff, crm, sfr, wbk.
  • Browser + Version: a combination of the browser class and the current version.

How to use it?

<!-- load jQuery -->
<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.5.1/jquery.min.js"></script>
<!-- load the plugin -->
<script type="text/javascript" src="jquery.detectBrowser.js"></script>
<!-- add the plugin to your existing ready() event -->
<script type="text/javascript">
$(document).ready(function() {
    // adding the classes into the <body> tag
    $("body").detectBrowser();
});
</script>
